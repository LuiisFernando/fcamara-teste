# fcamara-teste

test to FCamara.

To run follow the steps bellow.

1- install the dependencies

if you're using yarn 
```
yarn 
```
or npm

```
npm install
```

2- run the server with json server.

```
yarn json-server server.json -w -p 3333
```

3- run the project
```
yarn start
```

or

```
npm start
```
